# TestRail Export – Manual Test Cases

This folder contains manual test cases exported from **TestRail** in CSV format.  
These test cases were created for learning and practice purposes to demonstrate manual testing and test case design skills.

---

## Contents

- `saucedemo.csv` – Test Cases

---

## Test Case Details

The exported test cases include the following fields:

- Test Case ID  
- Title / Scenario  
- Preconditions  
- Test Steps  
- Expected Result  
- Priority  
- Test Type (Functional/UI)

**Note:**  
Test steps in the CSV file are exported from TestRail in HTML list format (`<ol><li>`). This is the default TestRail export format.

---

## Scope of Testing

The test cases cover:

- Positive test scenarios  
- Negative test scenarios  
- Edge cases  
- Input validation  
- Functional behavior
